# Customer Management System

This is a simple customer management system which allows you to manage customers and transactions. It also generates pdf receipts for the transactions.

## Features

* Manage customers
* Manage transactions
* Generate pdf receipts for transactions
* Search for customers and transactions
* View customer and transaction details
* Delete customers and transactions
* Sort customers and transactions by name, mobile number, transaction date, and transaction amount
* Filter customers and transactions by name, mobile number, transaction date, and transaction amount
* Export customers and transactions to csv

## Technologies Used

* Spring Boot
* Thymeleaf
* SQlite
* iText

## Set up

1. Clone the repository
2. Run the application using the command `mvn spring-boot:run`
3. Build using `mvn clean package -DskipTests`
